## 资源导入与引用的分层演进

### 第一层：首版 MVP

- URL 手动录入
- URL 自动补标题 / 资料概况 / 基础来源信息
- 本地 `txt / md` 上传
- 手动创建 `resource`
- 手动创建 `resource-fragment` 保留，但降级为补充入口
- 导入失败后的重试、手动补录与用户覆盖保护

### 第二层：自动化增强

- 导入后自动生成少量高置信度 fragment 候选
- 学习过程中自动创建 / 复用 fragment
- 更强的资料摘要、来源补全与候选去重
- fragment 合并 / 去重提示与引用校对

### 第三层：后续能力

- `PDF / DOCX` 导入
- 完整资料阅读器
- 引用对照区
- 外部资料检索

### 额外依赖与前置

- HTML 可读正文抽取策略与 URL 抓取 fallback
- 独立的资料正文基础 / 索引存储
- `PDF / DOCX` 解析器
- 更稳定的引用锚点与回跳方案
